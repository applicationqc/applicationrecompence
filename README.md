# Le P'tit Cash - Application de Jeu

Application de divertissement Flutter avec système de points virtuels.

## 🎮 Description

Le P'tit Cash est une application de jeu où les utilisateurs peuvent :
- Jouer à des mini-jeux amusants
- Collecter des PtitCoins (points virtuels)
- Débloquer des récompenses virtuelles
- Participer à des classements
- Compléter des missions quotidiennes

⚠️ **Important** : Les PtitCoins sont une monnaie virtuelle de jeu sans valeur monétaire réelle.

## 🚀 Fonctionnalités

- **Mini-Jeux** : Clicker, Memory, Cartes à gratter
- **Système de Progression** : Niveaux, XP, séries quotidiennes
- **Boutique Virtuelle** : Débloquez badges, avatars, thèmes
- **Classements** : Comparez vos scores avec d'autres joueurs
- **Firebase** : Authentification et sauvegarde cloud
- **AdMob** : Publicités intégrées
- **BitLabs** : Enquêtes partenaires

## 📋 Conformité

- **RGPD** : Politique de confidentialité complète
- **Google Play** : Conforme aux politiques de contenu
- **Données** : Authentification Firebase, progression de jeu
- **Publicités** : Google AdMob avec transparence
